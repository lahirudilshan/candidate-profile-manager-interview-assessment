## Installation

Preinstall tools

- yarn / npm
- git
- Docker (database container)

Clone the repository

```bash
  git clone https://github.com/lahirudilshan/candidate-profile-manager-interview-assessment.git
```

Run Docker for connect database

```bash
docker compose up
```

Install dependence

```bash
  yarn install
```

Run the application

```bash
  yarn run dev
```

Application live in

```bash
  http://localhost:3000/
```
